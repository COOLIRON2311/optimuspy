Требования, предъявляемые к программам
  • Исходные тексты программы не должны содержать функции с именем main
  • Программа не должна принимать аргументы командной строки
  • Выполнение программы должно осуществляться функцией с любым названием (кроме main), не принимающей параметров
  • В случае наличия в программе только одной функции, она используется для выполнения программы
  • В случае наличия в программе функции с именем optimus, она используется для выполнения программы независимо от наличия других функций
  • В остальных случаях при наличии в программе нескольких функций пользователю предлагается выбрать функцию вручную
